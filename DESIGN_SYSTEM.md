# HUD Design System

## Color

- HUD cyan: `#00D9FF`
- HUD cyan dim: `#087687`
- Deep black: `#020607`
- Glass fill: `rgb(0 217 255 / 0.06)`
- Glass line: `rgb(0 217 255 / 0.62)`
- Text primary: `#E8FBFF`
- Text muted: `#79E7F5`
- Alert amber: `#F4B340`

## Typography

- Display/UI: Space Grotesk
- Technical readouts: IBM Plex Mono
- Fallbacks: system sans and monospace stacks

## Effects

- Persistent black cockpit field
- Cyan holographic glow
- Glass panels with shimmer
- Fine scanlines
- Dotted coordinate grid
- Angular 45 degree clipped borders

## Motion

- Transitions: 220ms to 360ms
- Easing: cubic spring-like `cubic-bezier(0.2, 0.8, 0.2, 1)`
- Ambient animation only: radar sweep, holo rotation, shimmer, status pulse

## Spacing

- Base unit: 8px
- Compact gaps: 8px, 12px
- Component gaps: 16px, 24px
- Page padding: 24px mobile, 48px tablet, 72px desktop

## Components

- Cockpit shell
- Top coordinate bar
- System status widget
- Clock widget
- Radar widget
- Hologram widget
- Bottom navigation
- Mobile menu drawer
- Hero panel
- Project card
- Timeline card
- Blog card
- Terminal panel
- Hologram panel
